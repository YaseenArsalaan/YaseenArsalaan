# 🧠 LLM-Powered RAG Career Intelligence System

An AI-powered career assistant that simulates real interview scenarios, analyzes resumes, and matches job descriptions using LLM-based reasoning.

## 🚀 Features
- 💬 AI Interview Chat (answers like a recruiter)
- 📊 Resume Analyzer (ATS score + improvements)
- 🎯 Job Match System (match %, missing skills)
- 🔥 HR Killer Answers (perfect interview responses)

## 🧠 Tech Stack
- Python
- Streamlit
- OpenAI API (LLM)

## 💡 What This Project Shows
- Practical use of LLMs in real-world applications
- Prompt engineering for structured outputs
- Building intelligent career tools beyond static resumes

## ▶️ Run Locally
pip install streamlit openai  
streamlit run app.py
